◆ outer()

U8 N4VM::outer ( )

outer-interpreter

virtual machine execute single step (outer interpreter)

Returns
1: more token(s) in input buffer
0: buffer empty (yield control back to hardware)

console ok prompt

get a token from console

  • word address or numeric value

parse action from token (keep opcode in tmp)

immediate words,

  • : (COLON), switch into compile mode (for new word)
  • VAR, create new variable
  • CST, create new constant
  • PCI, create a pin change interrupt handler
  • TMR, create a timer interrupt handler
  • period in 0.1 sec
  • FGT, rollback word created
  • DMP, memory dump
  • RST, restart the virtual machine (for debugging)
  • BYE, restart

execute colon word (user defined)

execute primitive built-in word,

push a number (literal) to stack top,

or, error (unknown action)
